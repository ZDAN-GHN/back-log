## 项目介绍

基于 React + TypeScript 开发的个人代办记录小工具，使用 Cursor 快速构建原形，数据采用本地存储，无后端，开箱即用

## 快速开始

请确保 nodejs 版本不低于 20.18.0，在项目根路径下执行如下命令：

```
npm install
npm run dev
```
