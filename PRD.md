# 个人代办事项应用（web 应用） PRD

## 核心功能

### 1. 代办事项管理（CRUD）

- **创建任务**：支持输入任务标题、描述、选择分类、设置优先级（低、中、高）及截止日期。
- **编辑任务**：支持修改任务内容、截止时间、分类及优先级。
- **删除任务**：支持单条删除，删除时需二次确认（防误触）。
- **状态变更**：一键标记任务为“完成”或“未完成”。
- **任务排序**：
  - 按截止日期（升序/降序）
  - 按优先级（高到低）
  - 按创建时间（倒序）

### 2. 代办事项的数量统计

- **任务概览**：
  - 总任务数
  - 待完成数
  - 已完成数
- **进度可视化**：展示任务完成率（如环形进度条或线性进度条）。

### 3. 代办事项的分类管理

- **预设分类**：
  - 全部
  - 工作
  - 学习
  - 生活
- **自定义分类**：
  - 支持用户新增、修改、删除自定义分类。
  - 创建分类时支持选择预设的 **暖色系颜色**（如：蜜桃粉、柠檬黄、薄荷绿等）和 **图标**。

### 4. 代办事项搜索与筛选

- **搜索**：支持按任务标题和描述进行模糊搜索。
- **筛选**：
  - 按分类筛选（工作/学习/生活/自定义...）
  - 按优先级筛选（高/中/低）
  - 按完成状态筛选（全部/未完成/已完成）

### 5. 今日任务视图

- **今日聚焦**：仅展示截止日期为“今天”的任务。
- **过期提醒**：突出显示已过期但未完成的任务（标红或置顶），避免遗漏。
- **搜索支持**：支持在此视图下按关键词快速查找任务。

### 6. 设置与偏好

- **音效开关**：允许用户开启或关闭操作音效（点击按钮、完成任务时的“叮”声）。
- **主题设置**：默认 **暖色/可爱风** 主题，预留未来扩展夜间模式的接口。

## 设计要求

### 视觉风格

- **总体基调**：简洁现代，采用 **暖色调** 和 **可爱风**。
- **色彩运用**：使用柔和的马卡龙色系（Pastel Colors），避免高饱和度的刺眼颜色。
- **形状语言**：大量使用圆角（Rounded corners），卡片式设计，配合柔和的阴影（Soft shadows）增加层次感。
- **字体**：选用圆润、亲和力强的字体。

### 交互体验

- **流畅动画**：
  - 列表加载时的交错动画。
  - 添加/删除任务时的平滑过渡（Fade in/out, Slide）。
  - 完成任务时的愉悦动效（如：文字划线动画、粒子效果）。
- **反馈机制**：
  - **视觉反馈**：按钮点击态（缩放/变色）、输入框聚焦态。
  - **听觉反馈**：关键操作（添加、完成、删除）需伴随短促悦耳的音效（需提供开关）。

### 适配性

- **响应式设计**：完美适配移动端（Touch friendly）和桌面端。
- **移动端优化**：底部导航栏或易于单手操作的布局。
