# 个人代办事项应用（web应用） PRD

## 核心功能
1. 代办事项的数量统计
   - 总任务
   - 待完成
   - 已完成
   - 完成率
2. 代办事项的分类
   - 全部
   - 工作
   - 学习
   - 生活
   - 其他（支持用户自定义分类）
3. 代办事项搜索，支持多种类型的搜索
   - 内容
   - 分类
   - 优先级
4. 今日任务
   - 展示当日的所有任务
   - 支持具体搜索到某一个任务（按任务内容进行搜索）

## 设计要求
- 简洁现代的设计风格
- 暖色主题，可爱风
- 流畅的交互动画
- 点击按钮后需要有反馈（视觉 + 听觉）
- 移动端适配
