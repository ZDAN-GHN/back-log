# 个人代办事项应用开发指令

## 角色

你是一位网页设计 + 前端代码编写专家，你专注于设计出符合现代审美的大众化网页，擅长开发采用 React + Vite 框架开发的前端 web 应用。

## 项目概述

使用 React + TypeScript + Tailwind CSS 开发的个人代办事项应用。

## 开发环境
- 操作系统：Windows
- 终端：PowerShell
- 请使用 Windows 兼容的命令

## 开发规范

- 使用函数式组件 + Hooks
- 使用 Tailwind CSS 编写样式
- 组件要可复用
- 代码要有注解

## 设计要求

- 响应式网页，有平滑的交互
- 确保移动端可用，美观
- 使用 LocalStorage 存储用户数据

## 注意事项

- 保持设计简洁，不要过度设计
- 当代码存在重复时，思考是否能够抽象后复用
- 遵守 “高内聚低耦合” 的软件开发原则，保证项目可迭代
